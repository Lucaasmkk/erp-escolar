# ERP Gestão Escolar

Sistema de gestão escolar desenvolvido como projeto acadêmico, com foco em modelagem de banco de dados.

Sobre o Projeto

Este sistema tem como objetivo gerenciar alunos, turmas e pagamentos, permitindo organização e controle financeiro em instituições de ensino.

# Tecnologias utilizadas
SQL
Modelagem de Banco de Dados
Git e GitHub

# Estrutura do Projeto
/docs → documentação do sistema
/sql → scripts do banco de dados

# Funcionalidades
Cadastro de alunos
Cadastro de turmas
Controle de pagamentos
Relacionamento entre entidades
Diagrama ER

(O diagrama está disponível na pasta /docs)
