# ERP Gestão Escolar

Sobre o projeto

Este projeto tem como objetivo desenvolver a base de um sistema ERP escolar, voltado para a organização de informações acadêmicas como alunos, professores, turmas, notas e frequência. Nesta primeira etapa, o foco foi estruturar o banco de dados, pensando em como as informações seriam armazenadas e relacionadas.

Objetivo da Etapa 1

- Definir a estrutura inicial do sistema  
- Modelar o banco de dados  
- Criar as tabelas principais  
- Estabelecer os relacionamentos entre elas  
- Desenvolver o script SQL inicial  
- Criar o diagrama ER  

Estrutura do banco de dados

O sistema foi organizado com as seguintes entidades principais:

- Aluno  
- Professor  
- Disciplina  
- Turma  
- Matrícula  
- Notas  
- Frequência  

A tabela de matrícula foi utilizada para fazer a ligação entre aluno e turma, permitindo organizar melhor esse relacionamento.


 Tecnologias utilizadas

- SQL 
- GitHub  
- dbdiagram.io  

Estrutura do repositório

- /sql → scripts do banco de dados  
- /docs → documentação  
- /diagram → diagrama ER  


Status do projeto

Etapa 1 concluída. O projeto ainda está em fase inicial e será evoluído nas próximas entregas.

Observações

No começo tive um pouco de dúvida em como organizar os relacionamentos entre as tabelas, principalmente entre aluno e turma, mas com o uso da tabela de matrícula a estrutura ficou mais clara.
