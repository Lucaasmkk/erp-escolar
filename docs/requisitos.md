# Documento de Requisitos
Problema

Muitas escolas possuem dificuldade em gerenciar alunos, turmas e pagamentos de forma organizada.

Objetivo

Criar um sistema ERP para controle escolar.

Requisitos Funcionais
Cadastrar alunos
Cadastrar turmas
Registrar pagamentos
Vincular alunos às turmas Requisitos Não Funcionais
Sistema simples
Dados organizados
Segurança básica
